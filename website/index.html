free weather api based on https://api.met.no

        # I(Borislav Nikolov) got upset how people are selling free
        # data at some ludicrous prices, so I made freeweatherapi.com
        # using cached data from api.met.no

        # at the moment this is costing me 8€ for the domain and about
        # 5€ per day for digital ocean, I promise to run it until it
        # costs me < 50€ per month, if you dont like that just run it locally


        ## data is updated daily, and the lat/lng resolution is +-100km
        ## predictions are up to 1 month in the future, but you can
        ## also look at the past

        ## api - https://freeweatherapi.com/v1/query, method POST

        {
           "locations" : [
              {
                 "lat" : 0,
                 "lng" : -176,
                 "timestamp" : 1535284800
              }
           ]
        }
        lat: latitude
        lng: longituede
        timestamp: timestamp of the hour you are interested in
